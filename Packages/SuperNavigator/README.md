SuperNavigator
========

Sublime Text plugin to navigate you in current file.

![screencast](http://i.imgur.com/oHcFVSV.gif)

## Keymap

```javascript
[
    {
        "keys": ["super+k", "super+a"],
        "command": "super_navigate"
    },
    {
        "keys": ["super+k", "super+s"],
        "command": "super_navigate",
        "args": { "allTabs": true }
    }
]
```
